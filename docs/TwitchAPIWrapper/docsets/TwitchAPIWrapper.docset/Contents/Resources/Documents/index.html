TwitchAPIWrapper

Example

To run the example project, clone the repo, and run pod install from the Example directory first.

Requirements

Installation

TwitchAPIWrapper is available through CocoaPods. To install it, simply add the following line to your Podfile:

pod 'TwitchAPIWrapper'
